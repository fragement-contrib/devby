devby 开发辅助

简介

本项目用于提供前端开发常用功能，包括快速搭建开发调试服务器和一些常用的nodejs模块。

使用方式

你可以作为一个命令行使用，那么你需要全局安装： 

            npm install -g devby

也可以作为项目开发中的一个功能加强，在 package.json 中配置命令或nodejs文件中引入需要的模块使用，那么就在项目中执行安装命令： 

            npm install devby --save

命令行

全局安装后，就可以直接作为命令行使用了。你可以打印帮助查看： 

            devby -h

比如会出现下列内容： 

Usage: devby <command>

    where <command> is one of:
        --help, -h, --server, -s, --version, -v, --config, -c, --delete, --copy, --move, --network, --get, --post
    
    devby --help|-h <term>       search for help on <term>
    devby --help|-h              involved overview

比如--server或-s命令如何查看具体使用说明？只需要： 

            devby -h s

然后看见打印提示： 

            devby --server|-s [port|20000]         [2]render source server.

根据提示我们知道，这是一个快速启动一个本地资源服务器的命令，使用的时候可以设置服务器端口号，端口号缺省是20000。

比如我们启动端口为8080的服务器： 

            devby -s 8080

配置文件

对于更复杂的业务，我们推荐使用配置文件的方式。比如我们准备了配置文件 devby.config.js ，那么就可以： 

            devby -c ./devby.config.js

配置文件的内容如下： 

module.exports = {
    devServer: { // 服务器
        port: 8080, // 端口号
        proxy: [{ // 配置转发，可以任意多个
            test: /^\/example/, // 匹配规则
            target: "http://www.example.com" // 转发地址
        }]
    }
};

项目依赖

当然，这种方式我们还是可以在项目的 package.json 中通过命令进行使用，比如： 

{
    "scripts": {
        "dev": "devby -s 8080"
    }
}

不过，我们这里主要是描述可用模块有哪些。

命令行

参数解析

0.4.3 新增 
const { options } = require("devby");

const parsed = options(shortHands, argv);

例如： 

const parsed = options({
    '-v': '--version',
    '-h': '--help',
    '-s': '--server',
    '-c': '--config'
    }, process.argv);

返回的值parsed是一个简单可读的包含命令行所有信息的JSON，具体的你可以打印一下试试。

文件操作

删除

0.3.0 新增

删除文件或文件夹： 

const { deleteSync } = require("devby");

deleteSync(target);

复制

0.4.2 新增

复制文件或文件夹： 

const { copySync } = require("devby");

copySync(source, target);

移动

0.4.2 新增

移动文件或文件夹： 

const { moveSync } = require("devby");

moveSync(source, target);

打印

普通打印 

const { log, warn, error, linelog } = require('devby');

单行打印

0.4.3 新增 
const { linelog } = require("devby");

linelog(txt);

进度打印

0.4.3 新增 
const { process } = require("devby");

process(percentum, txt);
