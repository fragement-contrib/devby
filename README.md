# [devby](https://github.com/fragement-contrib/devby)
开发辅助命令

## 如何使用？

本项目用于提供更便捷的开发体验，功能会根据实际情况逐步调整。

```
npm install -g devby
```

然后，直接启动即可开启一个本地HTTP服务器：

```
devby
```

更多功能你可以打印帮助查看：

```
devby -h
```
